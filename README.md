# Task Manager CLI

Um gerenciador de tarefas simples baseado em linha de comando. Permite adicionar, listar, marcar/desmarcar e excluir tarefas.

## Funcionalidades

- Adicionar tarefas
- Listar tarefas
- Marcar/desmarcar tarefas como concluídas
- Excluir tarefas
- Persistência de dados em arquivo JSON
